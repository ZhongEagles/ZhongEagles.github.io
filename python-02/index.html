python-02 python扩展模块

重点

1、python3标准库的官方文档中文版：http://www.apiref.com/python-zh/library/index.html

定义

1）模块就是程序，每个扩展名为.py的Python程序都是一个独立的模块(Module)；

2）模块能定义函数、类和变量，让你能够有逻辑地组织你的Python代码；

3）组织模块，即包(package)，是放在一个文件夹里的模块集合；

模块引用方式

1）import <模块> [as <别名>]

　　将模块中的函数等名称导入当前程序 在调用模块中的函数的时候，需要加上模块的命名空间，可以给导入的命名空间替换一个新的名字。

2）from <模块> import <函数>

Python语言标准库

　　标准库是指在安装Python时就默认已经安装好的模块， Python具有功能强大的标准库；以下是有关标准库的一些常识。

扩展命名空间

命名空间(namespace)

1）表示标识符(identifier)的可见范围；

2）一个标识符可以在多个命名空间中定义，在不同命名空间中的含义互不相干；

　　不同命名空间内，即使名字相同，指向的数据对象也不同。

如何查看数据对象的内容

1、dir(<名称>)函数：列出名称的属性；

2、help(<名称>)函数：显示参考手册；
